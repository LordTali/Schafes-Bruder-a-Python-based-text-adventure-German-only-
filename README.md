Welcome to "Schafes Bruder" ("Sheep's Brother")!

Note: The software is only available in German.

This is a small text adventure game - developed in Python 3 - in which a simple peasant is on a quest to find his stolen sheep.
The game also serves to provide an entry to and as a test for my own roleplaying system called "Zeit der Zünfte" ("The Era of Guilds").

As the story progesses, players will be repeatedly prompted to type in a command. The game will recognise certain keywords and register a response.
The story will then react to the input and develop accordingly.

I will use the experience learned while making this game (and the feedback) to develop a more complex and ambitious game or another piece of software.

The code can also provide an entry for enthusiasts who are also learning to code and looking for an entry-level project sample.

With that being said: Happy coding and have fun with the game!
